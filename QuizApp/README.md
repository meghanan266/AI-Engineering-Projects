# AI Quiz App

A Blazor Server app that generates quiz questions using an LLM and grades your answers.

## How it works

- Questions on a chosen subject are generated by an `IChatClient` (Azure OpenAI / GitHub Models).
- Your answers are graded by asking the LLM whether they'd be accepted by a human examiner.
- The prompt is guarded against prompt-injection attempts in the answer text.

## Running locally

Set your AI endpoint and key via user secrets:

```
dotnet user-secrets set "AI:Endpoint" "https://SOMETHING.openai.azure.com/"
dotnet user-secrets set "AI:Key" "YourKeyHere"
```

Then run:

```
dotnet run
```
